Listas de nodos JavaScript HTML DOM

El objeto HTML DOM NodeList

Un objeto NodeList es una lista (colección) de nodos extraídos de un documento.
Todos los navegadores devuelven un objeto NodeList para la propiedad childNodes.
La mayoría de los navegadores devuelven un objeto NodeList para el método querySelectorAll().
EJEMPLO: El siguiente código selecciona todos los nodos <p> en un documento:

<!DOCTYPE html>
<html>
<body>
<h2>JavaScript HTML DOM</h2>
<p>Hello World!</p>
<p>Hello Norway!</p>
<p id="demo"></p>
<script>
const myNodelist = document.querySelectorAll("p");
document.getElementById("demo").innerHTML = "El innerHTML del segundo párrafo es: " + myNodelist[1].innerHTML;
</script>
</body>
</html>

Longitud de la lista de nodos HTML DOM

La propiedad length define el número de nodos en una lista de nodos:
La propiedad length es útil cuando desea recorrer nodos en una lista de nodos.
EJEMPLO: Cambiar el color del texto de todos los elementos <p> en una lista de nodos:

<!DOCTYPE html>
<html>
<body>
<h2>JavaScript HTML DOM</h2>
<p>Hello World!</p>
<p>Hello Norway!</p>
<p> Click en el boton para cambiar el color de todos los elementos p. </p>
<button onclick="myFunction()">Pruebalo</button>
<script>
function myFunction() {
const myNodelist = document.querySelectorAll("p");
for (let i = 0; i < myNodelist.length; i++) {
myNodelist[i].style.color = "red";
}
}
</script>
</body>
</html>
